SerialMock Serial Base Class¶

class serial_mock.Serial(stream)[source]¶
>>> Serial("COM99").MainLoop() # run forever on COM99

Serial(stream:string) instanciates a new MockStreamTunnel, stream should point to the comm port to listen on. in general this class should not be directly invoked but should be subclassed

Parameters:stream – a path to a pipe (ie “/dev/ttyS99”,”COM11”)
